# 🎯 Цель репозитория

Репозиторий содержит структурированные примеры анализа данных, алгоритмов машинного обучения и подходов к оценке качества моделей. Он используется как практическая база знаний для систематизации ключевых инструментов Data Science и хранения рабочих реализаций моделей

Это единое пространство для работы с данными, которое может использоваться как справочник при решении задач связанных с анализом данных

# 🗂 Структура репозитория

## 📂 01_data_analysis - работа с данными

Раздел состоит из 4 практических ноутбуков, которые покрывают полный цикл: от загрузки данных до подготовки признаков для модели.

### 01_data_visualization.ipynb — Первичный анализ данных (Pandas)

Практическая работа с табличными данными с использованием `pandas`.

В ноутбуке разбираются:

* Загрузка данных и первичная проверка структуры (`head`, `info`, `describe`)
* Индексация и фильтрация данных (логические условия, `.loc`, `.iloc`)
* Создание и удаление столбцов
* Работа с количественными и категориальными признаками
* Сортировка и агрегация
* Объединение таблиц:

  * `merge`
  * `join`
* Группировки и сводные таблицы:

  * `groupby`
  * `pivot_table`
  * `pd.crosstab`
* Практические задачи для закрепления навыков

Формируется уверенная база для работы с реальными датасетами.

### 02_eda_overview.ipynb — Exploratory Data Analysis (EDA)

Глубокий исследовательский анализ данных с акцентом на визуализацию и интерпретацию.

В ноутбуке:

* Анализ распределений количественных признаков
* Анализ категориальных признаков
* Исследование распределения целевой переменной
* Построение boxplot для поиска выбросов
* Матрица корреляций и анализ зависимостей
* Стилизация таблиц для улучшения читаемости
* Работа с геоданными:

  * визуализация через `Folium`
  * интерактивные карты (`Kepler`)
* Сравнительный анализ признаков относительно target

Фокус — научиться «читать данные» до построения модели.

### 03_data_quality_improvement.ipynb - Улучшение качества данных

Ноутбук посвящён повышению качества данных перед обучением модели.

Разбираются:

#### Работа с пропусками

* Удаление строк и столбцов
* Заполнение средним / медианой
* `KNNImputer`
* Indicator Method (добавление признака наличия пропуска)

### Работа с несбалансированными данными

* Анализ распределения классов
* Использование `imbalanced-learn`

### Работа с выбросами

* Межквартильный размах (IQR)
* Правило трёх сигм
* Визуальный анализ
* Статистические тесты

### Детекция аномалий

* `OneClassSVM`
* `Isolation Forest`
* `DBSCAN`
* `KNN-подход`

Показывается, как очистка и корректная предобработка напрямую влияют на стабильность модели.

### 4️04_model_quality_improvement.ipynb - Улучшение качества модели

Ноутбук посвящён работе с признаковым пространством для повышения качества модели.

#### Извлечение признаков (Feature Extraction)

* Работа с текстом (токенизация, нормализация)
* Извлечение признаков из дат
* Работа с геоданными

#### Кодирование категориальных признаков

* OneHotEncoding
* LabelEncoding
* Frequency / Count Encoding
* Target Encoding

#### Инженерия признаков (Feature Engineering)

* Логарифмирование
* Бинаризация
* Дискретизация
* Создание взаимодействий признаков

#### Отбор признаков (Feature Selection)

* Одномерные статистические методы
* RFE (Recursive Feature Elimination — рекурсивное исключение признаков)
* SelectFromModel
* Переборный подход

Фокус — системный подход к улучшению метрик модели через осознанную работу с признаками.


## 02_supervised_learning - обучение с учителем

> Задачи с целевой переменной

#### Регрессия

- Линейная регрессия
- Анализ коэффициентов модели
- Метрики качества:
  - MSE / RMSE
  - MAE
  - R²

#### Классификация

- Логистическая регрессия
- SVM (Support Vector Machine — метод опорных векторов)
- Метрики:
  - Accuracy
  - Precision / Recall
  - F1-score
  - ROC-AUC

### 03_unsupervised_learning - обучение без учителя 

> Анализ структуры данных без целевой переменной

- Кластеризация (K-Means, иерархическая)
- Снижение размерности (PCA — метод главных компонент)
- Интерпретация кластеров и компонент

### 04_ensemble_methods - ансамблевые методы машинного обучения

> Повышение качества за счёт комбинации моделей

Единый ноутбук с реализациями:

- Деревья решений
- Bagging (бэггинг — усреднение моделей)
- Random Forest (случайный лес)
- Boosting (градиентный бустинг)
- Voting / Stacking (комбинация предсказаний моделей)

Сравнение моделей и анализ метрик.

### 05_reinforcement_learning - обучение с подкреплением

> Базовые принципы Reinforcement Learning

- Формализация среды
- Агент, состояние, действие, награда
- Простейшие алгоритмы обучения
