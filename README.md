# Info

Централизованное хранилище моделей и инструментов для работы с данными и решения задач Data Science.

Репозиторий содержит структурированные примеры анализа данных, алгоритмов машинного обучения и подходов к оценке качества моделей.  
Используется как **практическая база знаний** и **портфолио**.

---

## 🎯 Цель репозитория

- Систематизировать ключевые инструменты Data Science
- Хранить рабочие реализации моделей и подходов
- Демонстрировать понимание полного DS-пайплайна:

`данные → анализ → модель → оценка → выводы`

- Использовать репозиторий как **reference (справочник)** и как часть резюме

## 🗂 Структура репозитория

### 01. Работа с данными

> Подготовка данных и первичный анализ

- Первичный анализ данных (EDA — exploratory data analysis, исследовательский анализ данных)
- Визуальный анализ и интерпретация распределений
- Улучшение качества данных:
  - обработка пропусков
  - работа с выбросами
  - приведение типов данных
  - предотвращение утечек данных (data leakage)

### 02. Обучение с учителем

> Задачи с целевой переменной

#### Регрессия

- Линейная регрессия
- Анализ коэффициентов модели
- Метрики качества:
  - MSE / RMSE
  - MAE
  - R²

#### Классификация

- Логистическая регрессия
- SVM (Support Vector Machine — метод опорных векторов)
- Метрики:
  - Accuracy
  - Precision / Recall
  - F1-score
  - ROC-AUC

### 03. Обучение без учителя

> Анализ структуры данных без целевой переменной

- Кластеризация (K-Means, иерархическая)
- Снижение размерности (PCA — метод главных компонент)
- Интерпретация кластеров и компонент

### 04. Ансамблевые методы машинного обучения

> Повышение качества за счёт комбинации моделей

Единый ноутбук с реализациями:

- Деревья решений
- Bagging (бэггинг — усреднение моделей)
- Random Forest (случайный лес)
- Boosting (градиентный бустинг)
- Voting / Stacking (комбинация предсказаний моделей)

Сравнение моделей и анализ метрик.

### 05. Обучение с подкреплением

> Базовые принципы Reinforcement Learning

- Формализация среды
- Агент, состояние, действие, награда
- Простейшие алгоритмы обучения

## 📌 Как использовать репозиторий

- Быстрый reference по моделям и подходам
- Основа для новых DS-проектов
- Демонстрация практических навыков для рекрутеров и команд
